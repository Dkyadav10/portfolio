Exploratory Data Analysis

Exploratory data analysis helps data analysts and scientists to become familiar with the data, make informed decisions about data preprocessing and feature engineering, and gain initial insights that can guide the subsequent steps of data analysis, modeling, and hypothesis testing. It is an iterative process that continues throughout the data analysis workflow, aiding in refining hypotheses and understanding the data from different angles. Some key aspects of EDA is data inspection, handling missing values, data visualization, data.
